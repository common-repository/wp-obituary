== Installation ==

1. Upload `WP_Obituary.zip` to the `/wp-content/plugins/` directory or Search WP Obituary Manager in you Wordpress admin dashboard add new plugin page 

2. Activate the plugin through the 'Plugins' menu in WordPress Dashboard 

3. Add your Obituary Default Thumnail and  Obituary  Default layout in   Obituaries ->  Settings 

4. Visit your Obituaries page to  www.yourdomain.com/obituaries 




== Frequently asked questions ==


- How can I add the Obituary?

You can add it dashboard under obituary postype. Or in obituary submit form in front end. 


- How can I access my obituary Page?

You can visit your obituaries page to  www.yourdomain.com/obituaries  



- How to manage consolences?

Condolences are manage in comments in Wordpress admin dashboard.
